# 📄 Multi-Agent RAG Scientific Paper Verifier

## 🚀 Overview

The Multi-Agent RAG Scientific Paper Verifier is an AI-powered system designed to analyze and verify scientific research papers. It uses a Multi-Agent architecture combined with Retrieval-Augmented Generation (RAG) to check claims, retrieve relevant information, and provide explanations with uncertainty analysis.

---

## 🎯 Features

* 🤖 Multi-Agent Architecture
* 🔍 Retrieval-Augmented Generation (RAG)
* ✅ Scientific Claim Verification
* 📖 Explanation Generation
* ⚠️ Uncertainty Detection
* 📊 Data Processing using CSV

---

## 🧠 Agents in the System

* **Retrieval Agent** – Fetches relevant information from data sources
* **Verification Agent** – Validates claims using retrieved data
* **Explanation Agent** – Provides human-readable explanations
* **Uncertainty Agent** – Identifies uncertain or weak claims

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/darpan1112/Multi-Agent-RAG-Scientific-Paper-Verifier.git
cd Multi-Agent-RAG-Scientific-Paper-Verifier
```

2. Install dependencies

```
pip install -r requirements.txt
```
